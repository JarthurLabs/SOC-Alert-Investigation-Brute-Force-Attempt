# Containment Plan

## Immediate Actions

1. Disable or temporarily lock the affected account.
2. Reset the user password.
3. Revoke active sessions.
4. Confirm MFA enrollment and recent MFA activity.
5. Review related accounts for password spraying.
6. Notify the account owner and manager.
7. Document all evidence.

## Follow-Up Actions

- Review login locations.
- Review mailbox rules if email account was involved.
- Confirm no unauthorized data access.
- Add or tune detection rule.
